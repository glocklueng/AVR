Enumeration Type Documentation

enum avrI2CReadWriteFlag

In 7-bit addressing procedure, the slave address is transferred in the first byte after the Start condition. The first seven bits of the byte comprise the slave address. The eighth bit is the read/write flag where 0 indicates a write and 1 indicates a read. I2C_READ is used in avr_i2c_read. I2C_WRITE is used in avr_i2c_write.
